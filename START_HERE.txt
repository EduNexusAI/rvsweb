⚙️ IF SOMETHING GOES WRONG
════════════════════════════════════════════════════════════════════════════

"npm not found"
  → Install Node.js from https://nodejs.org

"Port 3001 is in use"
  → Edit server.js and change port number
  → Or stop whatever's using port 3001

"Website won't load"
  → Make sure you run: npm start
  → Wait 2-3 seconds before visiting
  → Try refreshing the page
  → Check that URL is: http://localhost:3001 (not just localhost)

"Images not showing"
  → Check that assets/products/ folder has images
  → Check browser console for errors (F12)

"Forms not working"
  → Check email in assets/app.js
  → Check internet connection
  → Look for error messages in browser console
